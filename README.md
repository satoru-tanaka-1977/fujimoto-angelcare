# 時を紡ぐエンゼルケア

看護師・元美容師による、死に化粧（エンゼルケア）と
お見取り伴走サービスのランディングページです。

## 概要

Web制作の自己学習の一環として制作した架空クライアントのポートフォリオ作品です。
終末期医療・エンゼルケアという繊細なテーマに向き合い、
訪問者（ご遺族）の不安に寄り添うデザインと構成を意識して制作しました。

## 使用技術

- HTML5
- CSS3（カスタムプロパティ・Flexbox・アニメーション）
- JavaScript（Intersection Observer・ハンバーガーメニュー）

## 実装内容

- レスポンシブデザイン（スマートフォン対応）
- ハンバーガーメニュー（スライドアニメーション付き）
- スクロールフェードインアニメーション（Intersection Observer）
- CSSカスタムプロパティによるカラーシステム管理
- Google Fonts（Noto Serif JP / Noto Sans JP）

## 制作のこだわり

- 訪問者が緊急性の高い状況にあることを想定したUI設計
- 深い藤色・白磁・金色を基調とした和の格調あるカラーパレット
- 縦書きセクションラベルによる日本的な誌面構成
- 過度なアニメーションを避けた静かで厳かな表現
