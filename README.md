# 藤本真紀｜旅立ちの化粧とお見取りの伴走

## 概要

Web制作学習の一環として制作している架空のポートフォリオサイトです。

死に化粧（エンゼルケア）とお見取りをテーマに、
ご家族の悲しみに寄り添い、生前の尊厳を守ることをコンセプトとした個人サロン型サービスサイトを想定しています。

本プロジェクトは実在するサービスではなく、学習目的のデザイン・コーディング練習として制作しています。
